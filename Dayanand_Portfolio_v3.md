## 🎯 Make This Portfolio *Different* (High Priority for AI Agent)

This section defines **non-negotiable uniqueness principles**.  
The site should NOT feel like a template.

---

### 1. Presence > Information

The first screen (hero) should feel like:
> “Someone serious is here”

NOT:
- Resume dump
- Tech stack spam

**Implementation:**
- One strong line
- Subtle motion (breathing / glow)
- Minimal UI clutter

---

### 2. Systems Over Sections

Instead of:
- About
- Skills
- Projects

Structure like:
- Thinking
- Systems Built
- How I Approach Problems

**Goal:**
User should feel they are exploring a **mind**, not a profile.

---

### 3. Case Study > Project Cards

Avoid generic cards.

Each project should show:
- Problem
- Thought process
- Architecture
- Tradeoffs
- Outcome

Even 1 strong case study > 5 weak projects

---

### 4. Visual Thinking (Your Advantage)

You naturally think in diagrams.

**Must include:**
- System diagrams
- Flow representations
- Minimal but powerful visuals

This is your edge. Use it.

---

### 5. Controlled Motion

Animations should feel:
- Intentional
- Smooth
- Slightly slow

Avoid:
- Fast flashy transitions
- Random movement

Think:
> “calm power”, not “excited UI”

---

### 6. Dual Theme = Dual Personality

#### Dark Mode (Naruto — Fire / Action)
- Orange accents
- Strong contrast
- Slight glow
- Feels alive

#### Light Mode (Hyuga — Clarity / Awareness)
- Soft purple
- Clean whitespace
- Calm and precise

Switching theme should feel like:
> switching mindset

---

### 7. Depth Layers

Make UI feel layered:
- Background subtle gradients
- Foreground sharp content
- Hover = slight elevation

This creates **visual depth = mental depth**

---

### 8. Typography Discipline

- No fancy fonts
- Use spacing, not decoration
- Bigger headings, breathable layout

Clarity = intelligence

---

### 9. Interaction Philosophy

Every interaction should answer:
> “why does this exist?”

Examples:
- Hover reveals insight
- Click expands thinking
- Scroll reveals layers

---

### 10. Silence is Power

Don’t fill everything.

Empty space = confidence
